# AgriLens Usability Evaluation — System Usability Scale (SUS)

**Target participants:** 5–10 Egyptian farmers or agricultural students  
**Session format:** Think-aloud protocol, observed task completion  
**Related NFRs:** NFR-U1, NFR-U2, NFR-U3, NFR-U4, NFR-U5

---

## Task Scenarios (observe + time each)

| # | Task | NFR | Pass criteria |
|---|------|-----|---------------|
| T1 | Open the app, navigate to the camera, take a photo of a plant and submit a scan | NFR-U2 | Completed in ≤ 3 taps/steps |
| T2 | Find the result of your last scan and explain what the disease name and severity mean | NFR-U3 | Correct explanation without help |
| T3 | Add a new field named "Field A" with crop type Tomato | NFR-U1 | Completed without guidance |
| T4 | Locate the most recent notification and describe what it means | NFR-U4 | Correct interpretation |
| T5 | Switch the app language from English to Arabic | NFR-U5 | Completed, UI renders correctly in Arabic |

Record for each task: ✓ completed unaided / ✗ failed / time in seconds

## SUS Score Calculation

```
SUS Score = (
    (Q1 - 1) + (5 - Q2) + (Q3 - 1) + (5 - Q4) + (Q5 - 1) +
    (5 - Q6) + (Q7 - 1) + (5 - Q8) + (Q9 - 1) + (5 - Q10)
) × 2.5
```

| Score range | Adjective rating |
|-------------|-----------------|
| ≥ 85 | Excellent |
| 70–84 | Good |
| 50–69 | OK |
| < 50 | Poor |

**Thesis target:** SUS ≥ 70 (Good) to satisfy NFR-U1–U5.
